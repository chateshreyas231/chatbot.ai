# New Laptop Setup Guide

Complete guide for setting up a new company laptop.

## Initial Setup

### Step 1: Power On and Windows Setup
1. Connect power adapter
2. Power on laptop
3. Complete Windows initial setup:
   - Language and region
   - Accept license terms
   - Connect to Wi-Fi (Company-Guest or ethernet)
   - Sign in with company account

### Step 2: Install Company Software
IT should have pre-installed most software, but verify you have:
- Microsoft Office 365
- Microsoft Teams
- VPN Client
- Antivirus (Windows Defender or company standard)
- Browser (Chrome or Edge)
- Authenticator app

### Step 3: Configure Email
1. Open Outlook
2. Sign in with company email
3. Complete MFA if prompted
4. Wait for emails to sync

### Step 4: Set Up VPN
1. Open VPN client
2. Enter credentials:
   - Server: vpn.company.com
   - Username: your.email@company.com
   - Password: Your password
3. Connect and verify

### Step 5: Configure OneDrive
1. Sign in to OneDrive with company account
2. Choose folders to sync
3. Wait for initial sync to complete

## Security Setup

### Enable BitLocker (Windows)
1. Right-click Start > Settings
2. Update & Security > Device encryption
3. Turn on device encryption
4. Save recovery key securely

### Set Up MFA
1. Open Microsoft Authenticator app
2. Scan QR code from company portal
3. Test authentication

### Install Updates
1. Settings > Update & Security
2. Check for updates
3. Install all pending updates
4. Restart if required

## Software to Install

### Recommended (Optional)
- **Browser extensions**: LastPass, Grammarly
- **Development tools**: VS Code, Git (if developer)
- **Productivity**: Notion, Todoist
- **Communication**: Slack (if team uses it)

### Company Required
Verify these are installed:
- VPN client
- Antivirus
- Encryption software
- Company portal app

## Printer Setup

### Add Network Printer
1. Settings > Devices > Printers & scanners
2. Click "Add printer or scanner"
3. Select printer from list
4. Install drivers if prompted
5. Test print

### Troubleshooting
- If printer not found, check you're on company network
- May need IT to add printer permissions
- Contact IT for help with printer issues

## Wi-Fi Configuration

### Company Wi-Fi
- **SSID**: Company-Secure
- **Security**: WPA2-Enterprise
- **Username**: Your email
- **Password**: Your password
- **EAP type**: PEAP

### Guest Wi-Fi
- **SSID**: Company-Guest
- **Password**: Check company portal
- Use for initial setup only

## File Migration (If Applicable)

### From Old Laptop
1. Copy files to OneDrive or external drive
2. On new laptop, sign in to OneDrive
3. Files will sync automatically

### Backup
- All company files should be in OneDrive
- Personal files can be copied manually
- Verify all files transferred

## Final Checks

### Verify Everything Works
- [ ] Email sending/receiving
- [ ] VPN connects successfully
- [ ] Teams can join meetings
- [ ] OneDrive syncing
- [ ] Printer accessible
- [ ] Company portal accessible
- [ ] MFA working
- [ ] Software licenses activated

### Performance Check
1. Ensure all Windows updates installed
2. Check disk space (should have >50GB free)
3. Verify antivirus is running
4. Test all USB ports
5. Test camera and microphone

## Common Issues

### Issue: Can't Connect to Company Wi-Fi
**Solution**: May need certificate installation - contact IT

### Issue: OneDrive Not Syncing
**Solution**: 
1. Sign out and sign back in
2. Check available storage quota
3. Verify internet connection

### Issue: VPN Won't Connect
**Solution**: See VPN troubleshooting guide

### Issue: Missing Software
**Solution**: Contact IT or install from company software portal

## Return Old Laptop

If replacing an old laptop:
1. Back up all personal files
2. Remove personal accounts/apps
3. Factory reset if required
4. Return to IT department
