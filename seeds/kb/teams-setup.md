# Microsoft Teams Setup Guide

Complete guide for setting up and using Microsoft Teams for collaboration.

## Initial Setup

### Download Teams
1. Go to teams.microsoft.com/download
2. Download for your platform (Windows, Mac, iOS, Android)
3. Install the application

### First Sign-In
1. Open Teams
2. Enter your work email (firstname.lastname@company.com)
3. Enter password
4. Complete MFA if prompted
5. Teams will configure automatically

### Join Company Team
1. Click "Teams" in left sidebar
2. Click "Join or create a team"
3. Select "Company All Hands" or your department team
4. Click "Join team"

## Basic Features

### Chat
- Click "Chat" in sidebar
- Click "New chat" to start conversation
- Type @username to mention someone
- Use emojis and GIFs (click smiley face)

### Video Meetings
1. Click "Calendar" in sidebar
2. Click "New meeting" or "+"
3. Add participants
4. Set date/time
5. Click "Send"

### File Sharing
- In chat or channel, click paperclip icon
- Upload from computer, OneDrive, or SharePoint
- Files are automatically saved to SharePoint

### Screen Sharing
During a meeting:
1. Click share icon (bottom toolbar)
2. Choose screen, window, or whiteboard
3. Click "Share"
4. Click "Stop sharing" when done

## Troubleshooting

### Issue: Can't Sign In

**Solutions**:
1. Verify email and password are correct
2. Check internet connection
3. Try clearing Teams cache:
   - Close Teams completely
   - Delete: %APPDATA%\Microsoft\Teams (Windows)
   - Delete: ~/Library/Application Support/Microsoft/Teams (Mac)
   - Restart Teams

### Issue: Audio/Video Not Working

**Solutions**:
1. Check device permissions (Settings > Privacy > Camera/Microphone)
2. Test devices in Teams Settings > Devices
3. Ensure no other apps are using camera/mic
4. Update Teams to latest version
5. Check system audio settings

### Issue: Notifications Not Working

**Solutions**:
1. Open Settings (click profile picture > Settings)
2. Go to "Notifications"
3. Ensure notifications are enabled
4. Check system notification settings
5. Verify Do Not Disturb is off

### Issue: Meetings Lagging

**Solutions**:
1. Check internet speed (need minimum 1.5 Mbps)
2. Close unnecessary applications
3. Reduce video quality: Settings > Devices > Camera
4. Use wired connection instead of Wi-Fi
5. Check if VPN is interfering

## Best Practices

1. **Set status** - Let others know if you're available, busy, or away
2. **Use channels** - Post in relevant channels, not DMs for team-wide info
3. **React instead of reply** - Use emojis to acknowledge without cluttering
4. **Mute when not speaking** - Especially in large meetings
5. **Use @ mentions sparingly** - Only for urgent items or when needed

## Advanced Features

### Meetings Recording
- Start recording: Click "..." > "Start recording"
- Recordings saved to Stream/OneDrive
- Access via meeting chat or calendar invite

### Breakout Rooms
1. In meeting, click "..." > "Breakout rooms"
2. Create rooms and assign participants
3. Participants can join/leave rooms
4. Close rooms when done

### Live Transcripts
- Available in English meetings
- Click "..." > "Start transcription"
- Transcript appears in real-time
- Saved after meeting ends

### Whiteboard
- Click "..." > "Whiteboard" in meeting
- Draw, add sticky notes, collaborate
- Saved automatically to OneDrive

## Mobile App

### iOS/Android Features
- Same chat, meetings, and files as desktop
- Push notifications for messages
- Quick replies without opening app
- Join meetings from phone

### Tips
- Enable background app refresh
- Set up notification sounds
- Use mobile data for audio-only meetings
- Download files for offline access

## Need Help?

Common issues resolved:
- **Installation problems**: Download from official site, check system requirements
- **Login issues**: Verify credentials, check MFA
- **Missing teams**: Request access from team owner
- **Audio/video**: Check device permissions and settings

If issues persist:
- Create a ticket with error messages
- Include screenshots if possible
- Note when issue started
